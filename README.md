# 个人博客系统


基于Go语言和beego框架 开发的个人博客系统

## 编译安装说明：

1 . 设置GOPATH(安装目录)
    $ export GOPATH=/go/gocode/

2 . 下载安装
  $ go get github.com/lesroad/wblog

3 . cd 到 beego_blog 目录 执行
  main.go路径示例：go/gocode/src/beego项目/wblog/main.go
  $ bee run
